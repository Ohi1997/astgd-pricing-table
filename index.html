Installing Pricing Table

Download Design Folders

Download ASTGD Pricing Table from here. Unzip the downloaded folder and you will get 10 Design folders of the pricing table.

Inside every Design folder you can find 3 folders and 1 html file (index.html) . If you want to test the pricing table, we suggest you to open any Design folder with Visual Studio Code. Open the "index.html" file and run the file with "Open with Live Server" in any browser. If you wish switch into a different color or font, just navigate to js folder and look for the configsection.json file.

Here, you will find 10 colors and fonts are given as ColorComment & FontComment. Choose the color, font as your liking, set it to "themecolor" & "themefont" parameter respectively and save it to update the DOM.

Platform Implementation

ASTGD Pricing Table comes with different platform implementation. You can use this pricing table in WordPress/Laravel/CodeIgnitor sites as well. It can also be integrated into any existing website very easily.

HTML implementation

If you are creating a HTML based website or already have an existing HTML website. You can implement ASTGD Pricing Table very easily by following these steps.

Step One - Unzip the download pack. Inside there will be 10 Design folders. Choose the Design you like and open it with your code editor (VS Code Recommended);

Step Two - Take the chosen Design folder to your HTML file and insert the required css and js folders in your directory. If you have bootstrap and jquery already in your website, then please ignore "bootstrap.min.css" file from "css" folder and "jquery.min.js" file from "js" folder;

Step Three - Check the "index.html" file where you will find a "section" tag with the id "radioplans";

Step Four - Copy and Paste the section with id radioplans along with all it's classes to your desired location in your html file to use the pricing table.

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Your Website Title</title>
    <link rel="stylesheet" href="styles.css"> <!-- Link to your CSS stylesheet -->
</head>

<body>
    <header>
        <!-- Header content goes here: logo, navigation menu, etc. -->
    </header>

    <main>
        <!-- Main content of your webpage -->
        <div>
        <!-- Your content goes here -->
        </div>
        <section id="radioplans" class="tabs pricing special sec-normal bg-seccolorstyle">
            <!-- Pricing Table Content -->
        </section>
        <div>
            <!-- Additional content goes here -->
        </div>
    </main>

    <footer>
        <!-- Footer content goes here: copyright, contact info, etc. -->
    </footer>

    <script src="script.js"></script> <!-- Link to your JavaScript file -->
</body>
</html>

Step Five - Please check the path for the "configsection.json" file in "pricetable.js" file which is in "js" folder. Replace the "your/path/to/configsection.json" with your actual path of the json file. This path to the json file is very important to fetch value from json and update the DOM;

// Function to update pricing and handle visibility settings
function updatePricingAndSettings() {
    $.ajax({
       url: "your/path/to/configsection.json",
       dataType: "json",
       success: function (jsonData) {
         data = jsonData;
  
         // Update pricing
         var selectedheader = $("#headergroup li.active")[0].id;
         var defaultPeriod = getDefaultPeriodVisibility(data);
         updatePricing(selectedheader, defaultPeriod);
         currentPeriodHeader = defaultPeriod;
         resetPeriodHeader(defaultPeriod);
  
         // Handle visibility settings
         handleVisibilitySettings(data);
  
         // Update period headers
         updatePeriodHeaders(data);
       },
    });
 }

WordPress implementation

For WordPress enthusiasts venturing into website creation or those managing an existing digital platform, seamlessly adding an ASTGD Pricing Table is just a matter of following these intuitive steps.

Step One - We hope you already have WordPress installed on your localhost like XAMPP (i.e., your computer). Choose any design folder you like from ASTGD Pricing Table. Simply copy and paste 'css', 'js' and 'svg/image/png' folder to your localhost directory.

For this tutorial, We are using XAMPP, and the WordPress site is installed under the htdocs folder. Now, open XAMPP folder > htdocs > WordPress folder (in this case itâ€™s 'astgd-pricing-table'). Here, We have created a new folder 'pricetable' and moved all the folders and files from our chosen Design (in this case it's 'Design 1') .

Step Two - Next, We have added and activated two important WordPress Plugins so that we can implement our pricing table very easily. 1) Classic Editor and WP Code Insert Headers and Footers.

Step Three - Open the index.html file in any editor you like and only copy section div with the id "radioplans". Go to your Admin Dashboard and create a new page or open an existing page where you want to implement the pricing table. Now, add a block named "Custom HTML" and paste the plain html you just copied from index.html file.

Step Four - It's time to use the WP Code Insert Headers and Footers plugin to insert css links and js scripts into headers and footers. Make sure to add the correct path according to your pc's location. Then, click Save Changes button at the top right.

Step Five - Make sure to adjust the url path for configsection.json file to fetch data from the json. In your Chosen Design folder look for 'js' folder, Inside this folder open the pricetable.js in your code editor and simply update the correct path in the ajax method.

Step Six - Check all the changes you made and save it to update your page. Now, go to your page and you'll find ASTGD Pricing Table is implemented at your page.
